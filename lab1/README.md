# Лабораторная работа №1

## Тема

Сравнение алгоритмов сортировки.

## Реализованные алгоритмы

1. Bubble Sort — 3 балла
2. Insertion Sort — 5 баллов
3. Quick Sort — 10 баллов

Сумма: 18 баллов.

## Возможности программы

- ручной ввод последовательности;
- автоматическая генерация случайной последовательности;
- выбор алгоритма сортировки;
- вывод исходных и отсортированных данных;
- измерение времени выполнения;
- сравнение алгоритмов на одинаковых данных;
- тестирование алгоритмов.
