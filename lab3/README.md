# Лабораторная работа №3

## Тема

Введение в OpenMP. Простые примеры параллельных программ.

## Реализованные примеры

1. **Пример 1.1.** Hello World — каждый поток выводит свой идентификатор;
   главный поток сообщает общее количество потоков.
2. **Пример 1.2.** Параллельное скалярное произведение векторов через
   `#pragma omp parallel for reduction(+:sum)`.
3. **Пример 1.3.** Скалярное произведение, вынесенное в подпрограмму;
   используется директива `#pragma omp for reduction(+:sum)` внутри
   охватывающего `#pragma omp parallel`.
4. **Пример 1.4.** Поэлементное сложение векторов через
   `#pragma omp sections` (две независимые секции).
5. **Пример 1.5.** Умножение матрицы на вектор; параллельный цикл
   `#pragma omp for` с выводом промежуточных значений из критической секции.

## Запуск

```bash
cmake -S . -B build
cmake --build build
./build/openmp_intro_app 2     # 2 потока, интерактивное меню
./build/openmp_intro_tests     # автотесты корректности
```

В Docker:

```bash
docker build -t lab3 .
docker run --rm lab3
```

## Тесты

Тест проверяет, что значения, полученные параллельными версиями примеров
(скалярное произведение, поэлементное сложение, матрично-векторное
произведение), совпадают с эталонными.
