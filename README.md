# Мультипарадигмальное программирование — Sequence Lab

Сборник лабораторных работ по курсу «Мультипарадигмальное программирование».
Каждая работа размещена в отдельной папке `labN/` с независимой сборкой
через CMake и собственным `Dockerfile`. Лабораторные с №3 по №5 используют
OpenMP.

## Содержание

| № | Тема | Папка |
|---|------|-------|
| 0 | Базовая последовательность (`ArraySequence`, `ListSequence`) | [lab0](lab0/README.md) |
| 1 | Сравнение алгоритмов сортировки (Bubble / Insertion / Quick) | [lab1](lab1/README.md) |
| 2 | Алгоритмы поиска: бинарный поиск, BST, хэш-таблица | [lab2](lab2/README.md) |
| 3 | Введение в OpenMP: 5 примеров (`parallel`, `for`, `reduction`, `sections`, `critical`) | [lab3](lab3/README.md) |
| 4 | Релаксация Якоби в 3D-сетке: последовательная и OpenMP-версия | [lab4](lab4/README.md) |
| 5 | Параллельное умножение матриц на OpenMP | [lab5](lab5/README.md) |

## Требования к окружению

- Компилятор с поддержкой C++17 (`g++` 9+, `clang++` 10+, MSVC 2019+).
- CMake 3.16 или выше.
- Для `lab3`, `lab4`, `lab5` — OpenMP (входит в стандартный `g++`/`clang++`,
  для MSVC — флаг `/openmp`).
- Опционально Docker — у каждой работы есть свой `Dockerfile`.

## Универсальный порядок сборки

В каталоге любой лабораторной:

```bash
cmake -S . -B build
cmake --build build --config Release
```

После сборки в каталоге `build/` появляются исполняемые файлы вида
`<имя>_app` (демонстрационное приложение) и `<имя>_tests` (юнит-тесты).

## Отчёты

Подробные пояснения по каждой работе вынесены в PDF-файлы `ЛР_N.pdf`

реализации — `Lab_Report.pdf`.
